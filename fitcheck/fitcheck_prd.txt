# FitCheck! Product Requirements Document (PRD)

## Project Overview
- **Service Name:** FitCheck! - Fashion Power Meter
- **Target Audience:** Gen Z and Millennials (OOTD enthusiasts)
- **Design Concept:** Neo-Brutalism 'Hipster Label' style. Bold outlines, soft pastels, asymmetric shadows.
- **Core Principle:** Extreme simplicity. No login/signup.

## User Flow
1. **Main Screen:** One-touch photo upload + TPO input (Selectable).
2. **Loading Screen:** Fun fashion analysis animation.
3. **Result Screen:** 1:1 square card optimized for SNS sharing.

## Key Features (V1)
- **Sticker Overlay:** Client-side HTML5 Canvas API to overlay tier labels on uploaded photos.
- **Tier & Roasting:** 7 levels (Iron to Challenger) with humorous, sharp-tongued feedback.
- **Fashion Score:** Max 10,000 points, highly emphasized.
- **Angel & Devil Feedback:**
  - **Devil (😈):** Points to 'Worst' item, links to shopping malls (Musinsa/29CM).
  - **Angel (😇):** Points to 'Best' item.
- **5 Humorous Stats:**
  1. TPO Destruction
  2. Color Discord
  3. Eye Protection
  4. Confidence Max
  5. Wallet Defense

## UI/UX Constraints
- Single-page application feel (Pure HTML/CSS/JS).
- 3px bold black borders, 4px asymmetric shadows on all elements.
- Square-focused layouts.
